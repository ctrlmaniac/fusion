# Blockquotes

Without footer

```html
<blockquote class="blockquote">
  Lorem ipsum dolor sit amet consectetur adipisicing elit. Saepe, doloribus
  cupiditate consequuntur fugit, corporis, expedita repellendus vitae rerum
  ducimus minima aspernatur sequi culpa debitis. Facilis, excepturi dolores!
  Laboriosam, voluptatem architecto!
</blockquote>
```

With footer

```html
<blockquote class="blockquote">
  Lorem ipsum dolor sit amet consectetur adipisicing elit. Saepe, doloribus
  cupiditate consequuntur fugit, corporis, expedita repellendus vitae rerum
  ducimus minima aspernatur sequi culpa debitis. Facilis, excepturi dolores!
  Laboriosam, voluptatem architecto!

  <footer>lorem Ipsum <cite>dolor sit amet</cite></footer>
</blockquote>
```
